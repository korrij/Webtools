Who I Am

My name is Jessica Koroma. I am a passionate and creative learner who loves solving problems and building visually engaging websites. In this class, I learned to use HTML, CSS, and W3.CSS to build structured, styled, and responsive web pages. I enjoy applying design thinking and planning ahead to create a great user experience.

Technologies & Techniques Used

Throughout the project, I used semantic HTML5, external stylesheets, Flexbox, W3.CSS components, and media queries. I structured my pages using sections and div containers, created a responsive navigation menu, and applied consistent design patterns across all pages. I debugged and validated my code to ensure clean, accessible output.

How This Class Helped

This class gave me a strong foundation in web design and development. It helped me build confidence using visual layout tools and planning for responsive designs. The hands-on coding experience will support my professional goals of working in tech, especially in UX or front-end development.

Final Touches

I applied a soft and modern color palette using W3.CSS and custom CSS. The overall site has a clean layout and consistent style. I also ensured that every page links to this About Us page through the navigation bar.
